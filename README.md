<strong>OpenTournoi-Palet</strong>
<br>
<br>
<br>
Ce projet est une application web que vous pouvez héberger via votre propre serveur web (wamp, xamp, LAMP) pour une utilisation locale (déconseillée en ligne pour des raison de sécurité).
Les installeurs utilisent <a href="github.com/cztomczak/phpdesktop" target="_blank">php-desktop</a> pour embarquer l'application web dans un programme remplaçant le serveur web et lui donne une impression de logiciel portable.
<br>
<br>
<br>
Gestion d'un tournoi de palet vendéen.




- Gestion des équipes
- Gestion des phases de qualifications (nombre de phases à votre convenance) en mode aléatoire.
- Classement des phases qulificatives (Critères de tri paramétrable).
- Gestion des phases finales jusqu'à 128 équipes par phase, gestion des matchs en mode tableau et arbre de tournoi.
- Gestion des challenges à partir de 16 équipes par phase.
- Gestion des matchs de classement pour les phases à 8 et 16 équipes.
- Classement par phase finale.
- Classement général.
- Gestion des impressions, phase qualificatives, phases finales et classements.
- Générération d'un fichier index.html static pour la diffusion de votre tournoi en ligne, export vers un serveur FTP paramétrable (à partir de la BETA-0.16).
